This is a Machine Learning App for predicting heart failure ,The user is expected to input some values. some values have been encoded as follows: sex(male:1,female:0), RestingECG(Normal:1, ST:2, LVH:3), ExerciseAngina(NO:0,YES:1), ST_Slope(Up:1, Flat:2, Down:3), ChestPainType(ATA:1, NAP:2, ASY:3, TA:4)
